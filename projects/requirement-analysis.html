Project case study

Analysis of Predictive Policing Systems Using Chicago Data

Role: Researcher ¬∑ Tools: Python, Pandas, scikit-learn, Jupyter, GitHub

Summary: 10,000-word dissertation evaluating the effectiveness, bias, and ethical implications of predictive policing models using Chicago crime data.

Problem

Understand whether predictive policing tools genuinely improve crime prediction and how they might replicate historical bias.

Approach

Cleaned and modelled Chicago open crime data, compared baseline models with resource-allocation style models, and evaluated fairness across areas.

Outcome

Found that performance gains were tightly coupled to biased historical data, highlighting governance and deployment risks.
